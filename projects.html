人臉身分辨識整合系統
Identity Recognition
老師模型輸入影像特性: 輸入高解析影像，單一影像、多樣性小、鑑別性高，容易Overfitting.
學生模型輸入影像特性: 輸入低解析影像，多樣化影像，鑑別性低，容易Underfitting.
將兩者模型的特徵進行特徵融合，效果會較各別單一模型識別效果更佳，也驗證合成高解析影像識別結果叫低解析影像高。
"回歸損失函數"如何幫助合成以及識別
Regression Loss => Distribution
此圖為取模型中間特徵投影在2維的PCA空間上。 (左圖:沒有加上回歸損失函數) => 觀察到相同身分、不同解析度在空間中沒有聚類性。
(右圖:有加上回歸損失函數) => 觀察到相同身分、不同解析度在空間中有聚類性。
1. 有回歸損失時，低解析特徵會靠近高解析特徵，這可以讓合成品質更接近高解析影像。
2. 有回歸損失時，相同身分、不同解析度在空間上有聚類性，這可以幫助識別效果更好。

3. 推論時，引入Gan Inversion 技術，在特徵空間中動態尋找對應特徵，讓推論結果更好。

Neural network combining with clinical ultrasonography:Anewapproach for classification of salivary gland tumors

腫瘤資料來源
Resource data of tumor
- (A) 我們拿到原始的MRI腫瘤資料
- (B) 我們從(A)經過裁切後得到的腫瘤資料
- (C) (B)經過影像處理-直方圖均衡化得到的腫瘤資料
經過直方圖均衡化可以有效增加影像對比讓模型學習到更具有鑑別姓的特徵。

演算法架構圖
Type of Tumor Classification
1. VGG16 萃取影像特徵 2. 自定義全連階層並寫入Triplet loss (限制相同型態腫瘤在空間中接近，不同型態腫瘤在空間中遠離) 3. 自定義分類器進行訓練 4. 模組最後透過不同的機器學習模型進行Assemble learning. 透過Triplet loss得到初步較好的Boundary，在透過Assemble learning增加泛化性，讓測試資料準確率達100%。

PCB保護噴塗機

PCB保護噴塗機_流程圖細節
Flow
1.影像資料前處理:將影像轉換HSV色彩空間，針對使用者藍筆標記的保護區域進行二值化處理， 搭配形態學運算與遮罩生成濾除雜訊，提取目標區域，有效提升模型的學習專注度與後續分割準確度。
2.模型設計與規劃:採用YOLOv8-Segmentation進行 pixel-level分割，透過實驗發現:
高解析訓練的模型(pixel-size 小)：擅長精細輪廓處理，大區域或複雜圖形泛化能力較弱。
低解析訓練的模型(pixel-size 大)：適用大範圍、形狀複雜區域，對小區域邊界不精細。
透過實驗驗證整合兩模型輸出，達到補償效果並強化泛化能力，並克服初期資料不足(Few-shot learning)限制，有效縮短導入週期並突破製程瓶頸。
3. 路徑規劃演算法: 根據雙模型產出的遮罩結果，動態選取最佳遮罩進行處理，並以滑動視窗策略調整 X/Y 軸移動步長，規劃最適噴塗軌跡，自動生成控制序列串接設備執行。
4.系統整合與閉環訓練:開發半自動標註系統，將實際設備運行影像與標記資料回傳至伺服器，用於持續模型再訓練與更新，實現 AI系統的自我優化閉環架構。
5.實績成果： 成功導入產線流程，穩定執行區域分割與路徑生成任務，顯著提升設備運作精度與效率。整體控制誤差壓低至0~5mm，加快產線生產效能。

PCB外觀瑕疵檢測機

PCB外觀瑕疵檢測機_流程圖細節
Flow
1.虛實整合:解析PCB Gerber檔案以取得設計層之Pad與孔洞座標，結合實際設備拍攝光學影像，透過平面空間轉換將設計資訊對應至實際圖像，並完成目標區域裁切。
2.影像前處理: 針對文字圖樣進行前處理，包含二值化、Histogram 分析與形態學運算，強化切割準確性。因一個FOV無法涵蓋整板區域，嘗試使用Halcon進行影像拼接，取得完整、高解析輸入。
3. 模型設計與規劃:
Pad / Hole 缺陷檢測：採用 PatchCore無監督學習，僅需正常樣本即可建立基準分布，透過熱圖（Anomaly Heatmap）標示異常點，解決瑕疵樣本不足的問題。
文字鏡像判定：由於文字有群組印刷特性，先以影像處理技術分割單字，再以自建分類模型進行方向判斷，結合多數決機制（Majority Voting）決定整體鏡像與否。
外觀與線路瑕疵：採用 YOLOv10-OBB（Oriented Bounding Box）偵測旋轉型或不規則形狀之異常，強化對非預期形狀瑕疵的適應能力。
4.專案成果與效益：
高效能檢測指標：實測 Recall 達 0.95、Precision 為 0.9，為了降低誤判（漏檢），系統偏向Recall 優化，確保缺陷不漏掉，即使稍降低 precision，也大幅減少人工逐一檢查的負擔。
產線應用效益：全流程檢測時間僅約 15分鐘（PAD+Hole 約 15 分、文字鏡像與外觀各約 5 分，因多線程時間不累加)，取代人工需逐一檢查上萬個孔洞的不可能任務，大幅提升準確性與效率。
